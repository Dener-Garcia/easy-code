Conheça as principais tags HTML e para que elas servem

H1 - Título Principal

A tag H1 é a mais importante, servindo como o título principal da página. É essencial para SEO e para a organização do conteúdo.

<h1> Texto do Título </h1>
Praticar

H2 - Subtítulo

A tag H2 é usada para subtítulos, organizando o conteúdo em seções. Importante para hierarquizar informações e melhorar a navegação.

<h2> Texto do Subtítulo </h2>
Praticar

H3 - Subdivisão de Seção

A tag H3 é utilizada para subdividir seções definidas por H2, ajudando a detalhar ainda mais o conteúdo da página.

<h3> Texto da Subdivisão </h3>
Praticar

H4 - Subdivisão de Subdivisão

A tag H4 cria subseções dentro das seções definidas por H3, permitindo uma organização mais detalhada do conteúdo.

<h4> Texto da Subdivisão de Subdivisão </h4>
Praticar

H5 - Subdivisão Adicional

A tag H5 é usada para ainda mais subdivisões de conteúdo, sendo útil para uma estrutura hierárquica profunda.

<h5> Texto da Subdivisão Adicional </h5>
Praticar

H6 - Subdivisão Final

A tag H6 é a menor e mais específica subdivisão, ajudando a detalhar o conteúdo em níveis muito específicos.

<h6> Texto da Subdivisão Final </h6>
Praticar

p - Parágrafo

A tag p define um parágrafo de texto, separando o conteúdo em blocos de texto distintos e legíveis.

<p> Texto do Parágrafo </p>
Praticar

a - Link

A tag a cria links para outras páginas ou recursos. É essencial para navegação e conexões entre páginas.

<a href="url"> Texto do Link </a>
Praticar

img - Imagem

A tag img insere imagens na página. É usada para adicionar elementos visuais ao conteúdo.

<img src="url" alt="Descrição">
Praticar

figure - Figura

A tag figure é usada para agrupar conteúdo visual, como imagens, junto com uma legenda opcional usando a tag figcaption.

<figure><img src="imagem.jpg" alt="Descrição"><figcaption> Legenda </figcaption></figure>
Praticar

ul - Lista Não Ordenada

A tag ul cria uma lista não ordenada com itens de lista definidos por LI, útil para apresentar itens sem uma ordem específica.

<ul><li> Item </li></ul>
Praticar

ol - Lista Ordenada

A tag ol cria uma lista ordenada, onde os itens são numerados. Ideal para listar itens em uma sequência específica.

<ol><li> Item </li></ol>
Praticar

li - Item de Lista

A tag li é usada dentro de UL ou OL para definir cada item da lista. Ela estrutura os itens dentro de listas.

<li> Item </li>
Praticar

div - Divisão

A tag div é um contêiner genérico usado para agrupar outros elementos e aplicar estilos ou scripts.

<div> Conteúdo </div>
Praticar

section - Seção

A tag section define uma seção genérica de um documento, útil para dividir o conteúdo em partes temáticas.

<section> Conteúdo </section>
Praticar

span - Span

A tag span é usada para agrupar texto ou outros elementos em linha para estilização ou manipulação com CSS.

<span> Texto </span>
Praticar

article - Artigo

A tag article define um conteúdo independente, como um post de blog ou uma notícia, que pode ser distribuído e reutilizado.

<article> Texto </article>
Praticar

header - Cabeçalho

A tag header define uma seção introdutória ou cabeçalho de um documento, geralmente contendo títulos e navegação.

<header> Conteúdo </header>
Praticar

footer - Rodapé

A tag footer define o rodapé de uma página ou seção, geralmente contendo informações de contato e direitos autorais.

<footer> Conteúdo </footer>
Praticar

nav - Navegação

A tag nav é usada para definir seções de navegação dentro de um site, contendo links para outras páginas ou seções.

<nav> Links </nav>
Praticar

aside - Lateral

A tag aside define conteúdo lateral ou complementar, como barras laterais e informações adicionais relevantes.

<aside> Conteúdo </aside>
Praticar

form - Formulário

A tag form é usada para criar formulários, permitindo a coleta de dados dos usuários através de campos de entrada e botões.

<form> Campos </form>
Praticar

label - Rótulo

A tag label define um rótulo para um elemento de formulário, facilitando a acessibilidade e a associação entre rótulos e campos.

<label for="id"> Texto </label>
Praticar

input - Entrada

A tag input define campos de entrada dentro de um formulário, onde os usuários podem inserir dados como texto ou seleções.

<input type="text" />
Praticar

textarea - Área de Texto

A tag textarea define uma área de texto multilinha onde os usuários podem inserir uma quantidade maior de texto.

<textarea> Texto </textarea>
Praticar

select - Seleção

A tag select cria um menu suspenso, permitindo ao usuário escolher uma opção entre várias disponíveis que o desenvolvedor define através da tag option, ou sejam são utilizadas em conjunto.

<select><option value="Valor"> Opção </option></select>
Praticar

button - Botão

A tag button define um botão clicável que pode ser usado para enviar formulários ou executar ações JavaScript.

<button> Texto </button>
Praticar

table - Tabela

A tag table cria uma tabela para exibir dados em linhas e colunas, estruturando informações de forma tabular.

<table><tr><td> Dados </td></tr></table>
Praticar

iframe - Inline Frame

A tag iframe permite embutir outro documento HTML dentro do documento atual, útil para incorporar conteúdo externo.

<iframe src="https://example.com"></iframe>
Praticar

abbr - Abreviação

A tag abbr define uma abreviação ou sigla, proporcionando uma expansão opcional com o atributo TITLE.

<abbr title="Hypertext Markup Language">HTML</abbr>
Praticar

code - Código

A tag code é usada para exibir texto como código fonte, tipicamente dentro de blocos ou trechos de código.

<code> var x = 10; </code>
Praticar

blockquote - Citação

A tag blockquote é usada para citar longos trechos de texto, geralmente de outras fontes, com uma formatação especial.

<blockquote> Citação </blockquote>
Praticar

address - Endereço

A tag address é usada para fornecer informações de contato ou endereços relacionados ao conteúdo do documento.

<address> Contato: email@example.com </address>
Praticar

details - Detalhes

A tag details cria um widget interativo que permite ao usuário visualizar e ocultar informações adicionais ao clicar.

<details><summary> Título </summary> Conteúdo </details>

time - Tempo

A tag time é usada para definir datas e horários em um formato que pode ser facilmente interpretado por máquinas e buscadores.

<time datetime="2024-09-07"> 7 de setembro de 2024 </time>

data - Dados

A tag data é usada para associar valores de dados a uma representação de texto, útil para análise de dados e manipulação.

<data value="2024-09-07"> 7 de setembro de 2024 </data>

progress - Progresso

A tag progress exibe um indicador de progresso para uma tarefa, útil para mostrar o avanço de processos ou carregamentos.

<progress value="50" max="100"> 50% </progress>
Praticar

meter - Medidor

A tag meter exibe uma medição dentro de um intervalo conhecido, como progresso ou quantidades relativas.

<meter value="0.6" min="0" max="1"> 60% </meter>
Praticar

output - Saída

A tag output exibe o resultado de cálculos ou ações realizadas por scripts, útil para mostrar resultados interativos.

<output for="campo" name="resultado"> Resultado </output>
Praticar

svg - Gráficos Vetoriais

A tag svg é usada para criar gráficos vetoriais escaláveis e manipuláveis diretamente no HTML, ideal para gráficos e ícones.

<svg width="100" height="100"><circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" /></svg>
Praticar

ruby - Ruby

A tag ruby define anotações ou caracteres adicionais para textos, comumente usada para mostrar caracteres de ruby em textos em idiomas como o japonês.
